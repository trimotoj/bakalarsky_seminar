O práci

Cieľom bakalárskej práce je implementovať program na zarovnanie dvoch hudobných reprezentácií, napríklad partitúry a živej nahrávky tej istej skladby. Úlohou systému je určiť, ktorý úsek jednej reprezentácie zodpovedá rovnakému úseku druhej reprezentácie, a z tohto zarovnania odvodiť tempomapu hudobného záznamu.

Praktická časť zahŕňa spracovanie symbolických dát (napr. MusicXML, MIDI), extrakciu príznakov z audia, zarovnávanie sekvencií a vizualizáciu výsledkov. Web priebežne zhromažďuje literatúru, implementačné poznámky, screenshoty, ukážky kódu a týždenný denník postupu.

Zadanie – anotácia

Práca preskúma možnosti automatizovaného alebo čiastočne automatizovaného vytvárania tempomáp hudobných záznamov tak, aby hudobný záznam bolo možné zosúladiť s notovým materiálom.
